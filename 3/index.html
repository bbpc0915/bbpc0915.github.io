이웃 과 함께하는
선교적 공동체 가 되라!

  • 이제부터 기도제목을 따라 기도하세요.
  • 페이지 당 2~3분 정도 기도하시면 됩니다.

  1. 지역 사회에 선한 영향력을 끼치는 교회가 되도록


  2. 지역 사회의 필요를 돌아볼 줄 알고, 베풀어 주는 넉넉한 교회가 되도록


  3. 우리의 선한 행동과 섬김들이 교회 부흥으로 이어지도록


  4. 다가오는 주일에 있을 "사랑의 장기기증" 행사를 위해


  5. 선교를 위한 6가지 중점 사업(마을정화사업, 선한 사마리아인 프로젝트, 사랑의 장기기증, 무료 건강검진, 범천 커뮤니티 마켓, 청년 일자리 지원 프로젝트)이 모두 순조롭게 진행될 수 있도록









예배와 행사, 부르신 곳에서 응답


  1. 교회의 예배가 온전히 정상화 되도록


  2. 성도들이 예배의 가치를 깨닫고 예배를 통한 은혜를 경험하도록


  3. 예배의 순서를 맡은 자들과 주차, 방송, 안내, 발열 등으로 섬기는 자들을 위해


  4. 춘계심방 가운데 사랑과 은혜, 환대와 감사가 충만하도록


  5. 양육 훈련을 통해 많은 성도들이 말씀의 지식과 체험을 경험하고 성숙한 신앙인이 되도록









교역자&중직자, 그리스도의 종


  1. 담임목사님의 계속적인 성령 충만함과 영육의 강건함을 위해


  2. 교역자들이 맡은바 사명을 잘 감당하며, 가정의 평강을 위해


  3. 중직자들이 그리스도의 몸된 교회를 계속적으로 사랑하며 순종하도록


  4. 교역자들과 중직자들의 섬김을 통해 하나님의 은혜가 넘치는 교회가 되도록


  5. 춘계 심방을 잘 감당하게 하시고, 심방받는 성도들의 가정에 평강의 은혜가 임하도록







환우, 강건하길


  1. 육신의 질병과 고통이 환우들에게서 떠나가도록


  2. 정신적인 상처와 정서적인 아픔으로 힘들어하는 환우들이 하나님의 손길로 치유되도록


  3. 나음을 입은 자들의 질병과 고통이 재발하지 않도록


  4. 코로나에 감염되는 성도가 없게하시고, 코로나 백신을 맞은 성도들에게 부작용이 발생하지 않도록


  5. 환우들이 고통 가운데에서 더욱 하나님을 의지하도록








소그룹(구역/기관), 헌신과 섬김


  1. 소그룹 모임이 정상화, 활성화 되도록


  2. 구역장과 권찰, 기관장과 임원들이 맡겨진 사명 잘 감당하도록


  3. 구역 모임과 기관 모임에 감사와 기쁨이 넘쳐나도록


  4. 모이면 기도하고 교제하며, 흩어지면 전도하는 소그룹이 되도록







다음세대, 교회의 미래


  1. 유치부가 예수님의 사랑을 믿음으로 체험할 수 있도록


  2. 초등부가 말씀의 양식을 잘 받아 먹으며 그리스도 안에서 잘 자랄 수 있도록


  3. 중,고등부 세대가 방황하지 말고 비전을 향해 전진하도록


  4. 청년부 세대가 하나님을 의지하여 믿음으로 현실적인 문제들을 잘 관리하도록


  5. 교육 부서를 위해 보이지 않는 곳에서 수고하는 교사들과 봉사자들을 위해


  6. 어른들은 다음세대를 학대하지 않으며, 자녀들은 어른들을 공경하도록







나라/민족, 삶의 터전


  1. 우리 나라가 하나님의 평강의 은총을 누리도록


  2. 국가 위정자들이 하나님을 두려워하며, 정직하고 성실하게 일하도록


  3. 복음으로 거듭난 일군들이 사회적으로 큰 역할을 잘 감당하도록


  4. 한국 교회가 공공의 유익을 위하여 고민하며 기도하도록


  5. 코로나 방역이 잘 이루어지고, 집단 면역을 위한 백신 접종이 무탈하도록







전도/선교, 주님의 명령


  1. 부산범천교회 모든 성도들이 복음의 증인들이 되도록


  2. 복음으로 지역 사회를 돕고 섬기며 봉사하도록


  3. 선교사들을 통해 복음이 전해지는 곳곳마다 좋은 열매가 풍성히 열리도록


  4. 선교사들의 가정이 안전하고 건강하도록


  5. 후원하는 교회들이 성장하여 자립하도록


  6. 특별히, 미얀마와 적도기니가 나라의 평화를 되찾고 안정화 되도록
